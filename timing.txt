The R code "five" is:

(a) a numeric vector
(b) a complex vector
(c) a logical vector
(d) a character vector
Answer: d


R code to add 5 to each of the numbers 2, 4, and 6 would be:

(a) [ i+5 for i in (2,4,6) ]
(b) map(x->x+5, [2,4,6])
(c) c(2,4,6)+5
(d) add(2,4,6,amount=5)
Answer: c


Rows of a data frame matching a query can be retrieved using the function:

(a) filter
(b) select_from
(c) predicate
(d) search
Answer: a


Points can be drawn in a ggplot plot by adding:

(a) ggplot()
(b) geom_point()
(c) facet_grid(y ~ x)
(d) plot(x,y)
Answer: b


A region on a chromosome such as an exon would be represented in R using:

(a) a logical vector
(b) a GRanges object from package GenomicRanges
(c) an AAString object from package Biostrings
(d) a GTF object from package FASTA
